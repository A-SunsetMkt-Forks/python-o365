static oauth2(client_id, client_secret, store_token=True, token_path=None)[source]¶

Connect to office 365 using specified Open Authentication protocol

Parameters:
  • client_id – application_id generated by https://apps.dev.microsoft.com when you register your app
  • client_secret – secret password key generated for your application
  • store_token – whether or not to store the token in file system, so u don’t have to keep opening the auth link and authenticating every time
  • token_path – full path to where the token file should be saved to

Note

This method is part of fluent api and can be chained
