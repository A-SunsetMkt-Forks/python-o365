get_folders(limit=None, *, query=None, order_by=None, batch=None)[source]¶

Returns a list of child folders

Parameters:
  • limit – limits the result set. Over 999 uses batch.
  • query – applies a filter to the request such as “displayName eq ‘HelloFolder’”
  • order_by – orders the result set based on this condition
  • batch – Returns a custom iterator that retrieves items in batches allowing to retrieve more items than the limit.

get_messages(limit=25, *, query=None, order_by=None, batch=None, download_attachments=False)[source]¶

Downloads messages from this folder

Parameters:
  • limit – limits the result set. Over 999 uses batch.
  • query – applies a filter to the request such as ‘displayName:HelloFolder’
  • order_by – orders the result set based on this condition
  • batch – Returns a custom iterator that retrieves items in batches allowing to retrieve more items than the limit. Download_attachments is ignored.
  • download_attachments – downloads message attachments
